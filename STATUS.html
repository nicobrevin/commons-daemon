1. INTRODUCTION

The Daemon Component contains a set of Java and native code, including:

  • A set of Java interfaces applications must implement
  • Unix native code to control a Java daemon from a Unix operating system
  • Win32 code to control a Java daemon from a NT based Windows operating system

This code was originally developped as part of the Tomcat project.

2. DEPENDENCIES

The Daemon component is dependent upon the following external components for development and use:

  • Java Development Kit (Version 1.2 or later)
  • JUnit Testing Framework (Version 3.7 or later) - for unit tests only, not required for deployment
